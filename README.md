# LOL 中东服玩家社区门户 (LOL ME Server Community Portal)

汇聚中东服华人群友，一站式开黑工具与数据看板。

## 简介 (Introduction)

本项目是一个专门为《英雄联盟》中东服华人玩家设计的社区门户网站。它集合了新手指南、常用工具链接、语音开黑频道（Discord）以及群友排位数据排行榜等实用功能，旨在为玩家提供更加便捷的组队体验和社区交流环境。

## 核心功能 (Features)

*   **新手起航与客户端**: 提供中东服官方客户端下载链接及注册/转服教程。
*   **社群开黑语音**: 提供专属的 Discord 语音交流频道链接及语音优化辅助工具。
*   **游戏上分神器**: 推荐各种实用工具，如选人避坑神器 (HyperBoost Reveal) 及排队小游戏等。
*   **群友数据与排行榜**: 提供资料录入入口，以及中东服华人排行榜数据统计表格。

## 技术栈 (Tech Stack)

*   [Next.js](https://nextjs.org/) (App Router)
*   [React](https://reactjs.org/)
*   [Tailwind CSS](https://tailwindcss.com/)
*   [Framer Motion](https://www.framer.com/motion/) (动画)
*   [Lucide React](https://lucide.dev/) (图标)

## 本地开发 (Local Development)

```bash
# 安装依赖
npm install

# 运行开发服务器
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 即可预览。

## 部署 (Deployment)

推荐使用 [Vercel](https://vercel.com/) 进行部署，支持与 GitHub 仓库无缝集成。
